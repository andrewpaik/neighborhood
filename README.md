# Neighborhood

A social community platform connecting locals through group activities. Discover events, join groups, and build meaningful connections in your area.

## Features

- **Real-time Group Chat & Event Coordination** — Communicate instantly with community members and organize activities together
- **Location-Based Community Discovery** — Find groups and events happening near you with smart location filtering
- **Firebase-Powered Authentication & Data** — Secure sign-in and real-time database for seamless user experience
- **Activity Feed** — Stay updated on what's happening in your neighborhood

## Tech Stack

| Technology | Purpose |
|-----------|---------|
| Next.js | React framework with SSR and routing |
| Firebase | Authentication, Firestore database, and hosting |
| Tailwind CSS | Utility-first styling for rapid UI development |
| TypeScript | Type-safe development across the codebase |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/andrewpaik/neighborhood.git
cd neighborhood

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your Firebase config to .env.local

# Run the development server
npm run dev
```

## How It Works

1. **Sign up & set your location** — Create an account and set your neighborhood
2. **Discover communities** — Browse local groups based on your interests
3. **Join & participate** — RSVP to events and chat with group members
4. **Create your own** — Start a new group or organize an event for your community
